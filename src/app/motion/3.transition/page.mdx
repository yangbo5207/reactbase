## 支持动画的属性

Motion 可以为任何 css 属性值实现动画。其中包括大量 css 动画本身已经支持的属性。还包括 css 不支持的属性。例如 `mask-image`

我们可以为属性值传入

- 1、数字: `0`, `100`
- 2、包含数字的字符串: `0vh`, `10px`
- 3、颜色: 十六进制、RGBA、HSLA
- 4、包含多个数字/颜色的复杂字符串，例如 `box-shadow`
- 5、display: none/block
- 6、visibility: hidden/visible
