## 使用激活码的原因

原本计划在 Supabase 中设计新的 VIP 系统。

目前该计划的所有功能已经开发完毕，具体的流程是付费之后申请成为付费用户，由管理员审核确认之后获得阅读权限。

但是由于在开发过程中发现 Supabase 的访问速度偏慢，在权限验证的过程中会极大影响阅读体验，因此暂停了这个过程。

为了能够让已经付费的用户尽快阅读到付费内容，暂时选择使用激活码来确保付费用户的阅读权益。
