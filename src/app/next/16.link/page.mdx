## Link

我们可以使用 NextJS 内置的 Link 组件跳转到一个指定的路由。默认情况下，当 Link 出现在视口时，该路由对应的组件 RSC 将会被提前加载。

基础的用法如下

```tsx page.tsx
import Link from 'next/link'

export default function Page() {
  return <Link href="/dashboard">Dashboard</Link>
}
```
